# STRUCTURE — 으나수 네일 샘플

```
eunasoo-nail-sample/
├─ index.html            단일 페이지 (원페이지 브로슈어)
├─ CHANGELOG.md
├─ STRUCTURE.md
└─ img/
   ├─ logo.png            로고 전체(마크+워드마크), 투명 배경
   ├─ logo-mark.png       마크만 — 상단바·파비콘
   ├─ logo-mark-light.png 마크 라이트 퍼플 — 다크 푸터용
   ├─ hero.jpg            히어로 (펄 크롬 뉴트럴)
   ├─ band.jpg            소개 섹션
   ├─ archive.jpg         컬러 팁 진열 밴드
   └─ work-01 ~ work-12.jpg  갤러리 12컷
```

## 섹션 순서 (index.html)
1. 고정 상단바 — 로고 / 메뉴(데스크톱) / 예약 문의 버튼
2. **Hero** — 비대칭 스플릿. 좌: 로고+헤드라인+CTA 2개 / 우: 풀블리드 사진
3. **Marks** — 신뢰 3항목(완전 예약제 · 상태 확인 후 시작 · 유지력 우선). 수치 주장 없음
4. **About** — 소개 (사진 + 2문단 + 서명 라인)
5. **Menu** — 시술 5종 가격표 (라인 기반, 박스 없음)
6. **Works** — 갤러리 12컷 (모바일 2열 / 데스크톱 4열)
7. **Archive** — 다크 밴드. 컬러 팁 진열 + 색 고르는 법
8. **Care** — 시술 후 안내 4항목
9. **Notice** — 예약금 / 변경·취소 / 지각·노쇼 정책
10. **FAQ** — 5문항 `<details>`
11. **Visit** — 주소·시간·전화·주차 + 채널 4개(DM·전화·지도·인스타)
12. 푸터 (다크) + 모바일 하단 퀵바(전화 / DM 예약)

## 티어 판정
- **베이직(보여주기)** — 폼 없음. 예약은 인스타 DM·전화로 유도.
- 스탠다드로 올릴 경우: 예약 문의 폼(Formspree) + 네이버 예약 연동 추가.

## 커스터마이즈 포인트
- 색: `:root`의 `--plum`(#693B84) 하나만 바꾸면 액센트 전체가 따라감
- 갤러리 추가: `.grid` 안에 `.gcell` 블록 복제 + `img/work-NN.jpg` 추가
- 폰트: `--en`(Jost) / `--kr`(Pretendard). 납품 확정 시 **self-host 전환 필요**(현재 CDN)
